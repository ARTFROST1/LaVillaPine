# Assets Folder

This folder contains all images and media files for the La Villa Pine website.

## Folder Structure

- `images/` - Main website images
  - `hero/` - Hero carousel images  
  - `gallery/` - Gallery photos
  - `about/` - About page images
- `logos/` - Company logos and branding

## How to Upload Your Photos

1. **Using Replit Interface:**
   - Click the "+" button next to files
   - Select "Upload file" or "Upload folder" 
   - Choose your images from your computer

2. **Drag & Drop:**
   - Simply drag your image files from your computer
   - Drop them into the appropriate folder in the file explorer

3. **Organize by Category:**
   - Put hero carousel images in `assets/images/hero/`
   - Put gallery photos in `assets/images/gallery/`
   - Put logos in `assets/logos/`

## Supported Formats
- JPG/JPEG - Best for photos
- PNG - Best for logos with transparency
- WebP - Modern format for better compression
- SVG - Vector graphics for logos

## Recommended Sizes
- Hero images: 1920x1080px or larger
- Gallery images: 800x600px minimum
- Logos: SVG format preferred, or PNG with transparency

Once you upload your images, I'll update the website code to use your actual photos instead of the placeholder images.
